📚 Design System Diaspo'Actif

Documentation technique complète des composants, tokens et patterns de la plateforme.

ℹ️
Comment utiliser ce Design System
Incluez /assets/ds.css et /assets/ds.js dans chaque page. Tous les composants sont disponibles via les classes CSS da-* et l'API JavaScript window.DA.*.

⚡ API JavaScript (window.DA)

MéthodeDescriptionExemple
DA.openModal(id)Ouvrir une modaleDA.openModal('mon-modal')
DA.closeModal(id)Fermer une modaleDA.closeModal('mon-modal')
DA.confirm(opts)Boîte de confirmation (Promise)await DA.confirm({title:'…'})
DA.toastSuccess(msg)Toast succèsDA.toastSuccess('Sauvegardé !')
DA.toastDanger(msg)Toast erreurDA.toastDanger('Erreur')
DA.toastWarning(msg)Toast avertissementDA.toastWarning('Attention')
DA.toastInfo(msg)Toast informationDA.toastInfo('Info')
DA.createTable(opts)Tableau dynamique avec tri/filtre/paginationDA.createTable({containerId, columns, data})
DA.showContextMenu(e, items)Menu contextuel au clic droitel.oncontextmenu = e => DA.showContextMenu(e, items)
DA.fmtDate(iso)Formater une dateDA.fmtDate('2026-06-26') → '26 juin 2026'
DA.relTime(iso)Temps relatifDA.relTime(date) → 'Il y a 2h'
DA.avatarHtml(name, size)Avatar placeholder HTMLDA.avatarHtml('Jean D', 40)
DA.badgeHtml(text, type)Badge HTMLDA.badgeHtml('Actif', 'success')
DA.skeleton(count, type)HTML skeleton loaderDA.skeleton(3, 'card')
DA.buildBreadcrumb(id, items)Construire un fil d'ArianeDA.buildBreadcrumb('bc', [{label,href}])

📦 Intégration dans une page

<!-- Dans le <head> -->
<link rel="stylesheet" href="/assets/ds.css">

<!-- Avant </body> -->
<script src="/assets/ds.js"></script>

Compatibilité
Chrome 90+, Firefox 88+, Safari 14+, Edge 90+. Responsive de 320px à 2560px+. Compatible PWA et mode impression.
